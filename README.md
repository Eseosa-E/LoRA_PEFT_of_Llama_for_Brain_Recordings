# LoRA_PEFT_of_Llama_for_Brain_Recordings
This project applies Low-Rank Adaptation (LoRA) — a parameter-efficient fine-tuning  technique — to align LLaMA with fMRI brain recording data. The model is trained to  reconstruct natural language from neural signals recorded while participants listened  to spoken narratives, using a fraction of the parameters required by full fine-tuning.
